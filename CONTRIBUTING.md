# Contributing

Thanks for contributing to Paletto.

## Prerequisites

- Node.js >= 18
- pnpm >= 9

## Setup

```sh
pnpm install
```

## Common commands

- pnpm dev
- pnpm build
- pnpm lint
- pnpm check-types
- pnpm format

## Guidelines

- Keep changes scoped to the smallest possible area.
- Prefer adding types for any new API response.
- Update docs when behavior changes.

## Pull requests

- Describe the change and why it is needed.
- Include screenshots or recordings for UI changes.
- Note any follow-up work in the PR description.
