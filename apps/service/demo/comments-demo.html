Comments Demo

Create comments (drafts), list approved comments publicly, and approve or reject via owner or admin endpoints.

Create Comment

Requires auth token. New comments start as draft.

List Comments (Public)

Requires targetType + targetId. Only approved comments are returned.

Update Comment Status (Owner)

Requires auth token for the artwork/blog owner.

Admin List Comments

Requires admin token.

Admin Update Comment Status

Requires admin token.
